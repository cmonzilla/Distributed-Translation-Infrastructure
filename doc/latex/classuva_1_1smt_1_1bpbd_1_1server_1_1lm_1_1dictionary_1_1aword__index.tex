\subsection{Detailed Description}
This abstract class is used to represent the word dictionary. It contains no specific implementation but is more of an interface. It is used to allow for more word dictionary/index implementations.

Any implementation of this class must issue the unknown word $<$unk$>$ index 1 (U\+N\+K\+N\+O\+W\+N\+\_\+\+W\+O\+R\+D\+\_\+\+I\+D).

The first real word index will be therefore 2 (M\+I\+N\+\_\+\+K\+N\+O\+W\+N\+\_\+\+W\+O\+R\+D\+\_\+\+I\+D).

The issued word ids must be continuous and non-\/repeating, unique!

N\+O\+T\+E\+: All of the methods are non-\/virtual for the sake of avoiding virtual method call overheads!
