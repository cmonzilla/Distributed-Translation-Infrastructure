\subsection{Detailed Description}
Holds a buffer that cannot be modified. 

The \hyperlink{classasio_1_1const__buffer}{const\+\_\+buffer} class provides a safe representation of a buffer that cannot be modified. It does not own the underlying data, and so is cheap to copy or assign.

\begin{DoxyParagraph}{Accessing Buffer Contents}

\end{DoxyParagraph}
The contents of a buffer may be accessed using the \hyperlink{group__buffer__size}{asio\+::buffer\+\_\+size} and \hyperlink{group__buffer__cast}{asio\+::buffer\+\_\+cast} functions\+:
