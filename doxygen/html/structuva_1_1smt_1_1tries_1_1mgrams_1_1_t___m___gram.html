Detailed Description

This structure is used to store the N-Gram data of the back-off Language Model.

Parameters
probstores the log_10 probability of the N-Gram Must be a negative value
back_offstores the log_10 back-off weight (probability) of the N-gram can be 0 is the probability is not available
contextstores the n-gram's context i.e. for "w1 w2 w3" -> "w1 w2"
tokensstores the N-gram words the size of this vector defines the N-gram level.
levelstores the number of meaningful elements in the tokens, the value of N for the N-gram
