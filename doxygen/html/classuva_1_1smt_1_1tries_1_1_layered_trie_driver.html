This is the common generic trie base class for layered tries.

Compute the context hash for the M-Gram prefix, example:

N = 5

0 1 2 3 4 w1 w2 w3 w4 w5

contextLength = 2

0 1 2 3 4 w1 w2 w3 w4 w5 ^ ^ Hash will be computed for the 3-gram prefix w3 w4.

Parameters
is_back_offis the boolean flag that determines whether we compute the context for the entire M-Gram or for the back-off sub-M-gram. For the latter we consider w1 w2 w3 w4 only
querythe query state object
ctx_id[out] the context id to be computed
Returns
the true if the context could be computed, otherwise false
Exceptions
nothing
