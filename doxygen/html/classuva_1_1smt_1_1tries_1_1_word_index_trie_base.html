Detailed Description

template<TModelLevel N, typename WordIndex>
class uva::smt::tries::WordIndexTrieBase< N, WordIndex >

This is a common base class for all Trie implementations. The purpose of having this as a template class is performance optimization.

Parameters
N- the maximum level of the considered N-gram, i.e. the N value
