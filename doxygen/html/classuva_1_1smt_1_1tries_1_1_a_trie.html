Detailed Description

template<TModelLevel N>
class uva::smt::tries::ATrie< N >

This is a common abstract class for all possible Trie implementations The purpose of having this as a template class is performance optimization. It is a template class that has two template parameters:

Parameters
N- the maximum level of the considered N-gram, i.e. the N value
doCache- the indicative flag that asks the child class to, if possible, cache the queries.
