88  //Stores the maximum number of bits up to and including M-grams
89  //of level 5. We use sizeof (TShortId) as each wordId is of type
90  //TShortId, and the maximum number of bits is thus defined by the
91  //number of word_ids in the M-gram and their max size in bytes.
92  static constexpr uint8_t M_GRAM_MAX_ID_LEN_BYTES[] = {
93  0, 0,
