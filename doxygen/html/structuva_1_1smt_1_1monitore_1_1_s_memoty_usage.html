Detailed Description

This structure stores the memory statistics. Resident Set Size: number of pages the process has in real memory. This is just the pages which count toward text, data, or stack space. This does not include pages which have not been demand-loaded in, or which are swapped out. For more information see http://man7.org/linux/man-pages/man5/proc.5.html
