This function will combine two word references to get one hash map N-gram level reference. This is a Szudzik's function used for pairing. http://szudzik.com/ElegantPairing.pdf This function is more efficient that than of Cantor as it uses indexes densely the result of szudzik(N,M) is <= 2*max(M,N), so the risk of owerflows is much smaller!

xthe key word reference
ythe previous context
Returns
the context reference for the next N-gram level
