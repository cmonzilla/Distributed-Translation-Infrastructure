Detailed Description

template<typename trie_type, typename reader_type>
class uva::smt::bpbd::server::lm::arpa::lm_basic_builder< trie_type, reader_type >

This is the Trie builder class that reads an input file stream and creates n-grams and then records them into the provided Trie. This is an ARPA format based trie builder, so it expects that the provided model file contains a basic text model in ARPA format.
