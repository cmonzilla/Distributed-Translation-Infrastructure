class uva::utils::containers::DynamicStackArray< ELEMENT_TYPE, IDX_DATA_TYPE, INITIAL_CAPACITY, DESTRUCTOR >

This class represents a dynamic memory array and stores the main methods needed for its operation

Parameters
ELEMENT_TYPEthe array element type
IDX_DATA_TYPEthe type is to be used for the size, capacity and index variables, should be an unsigned type!
INITIAL_CAPACITYthe number of words, which defines the initial capacity.
DESTRUCTORthe destructor function to be used on the elements when the container is deleted, default is NULL
