Detailed Description

This basic storage class that stores the pointer to pre-allocated memory plus the length of this piece. This class can be used to represent a piece of text, a line or a word or any arbitrary piece of memory. Note that, the string here is not necessarily \0 terminated and the text memory can be Gb large! Also the memory is not managed by the class.
