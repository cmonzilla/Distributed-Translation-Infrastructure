Detailed Description

This is a hashing word index, it is trivial - each word gets an id which is its hash value. This also means that any word is considered to be a known word. Therefore, in the Tries if the word id has no associated payload then an unknown word payload is to be used. Still the unknown and undefined word ids are reserved nd should not be issued.
