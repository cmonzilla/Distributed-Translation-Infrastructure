virtual void uva::smt::bpbd::server::lm::proxy::lm_slow_query_proxy::execute ( text_piece_reader &  line)
pure virtual

Allows to execute m-gram the query. The query starts with the m-gram size given by min_level and then grows until the maximum of LM_M_GRAM_LEVEL_MAX. After that m-grams of the LM_M_GRAM_LEVEL_MAX are computed via a sliding window: Let: "min_level == 2", "LM_MAX_QUERY_LEN = 4", "num_word_ids == 6" and "word_ids == w1w2w3w4w5w6" Then this method will compute the sum: P(w2|w1) + P(w3|w1w2) + P(w4|w1w2w3) + P(w5|w2w3w4) + P(w6|w3w4w5)

Parameters
linethe text piece reader storing the m-gram query line
