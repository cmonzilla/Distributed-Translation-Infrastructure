\subsection{Detailed Description}
Class to inform the \hyperlink{classasio_1_1io__service}{io\+\_\+service} when it has work to do. 

The work class is used to inform the \hyperlink{classasio_1_1io__service}{io\+\_\+service} when work starts and finishes. This ensures that the \hyperlink{classasio_1_1io__service}{io\+\_\+service} object\textquotesingle{}s \hyperlink{classasio_1_1io__service_ac84bed0d1dd061bc71010ba1228439da}{run()} function will not exit while work is underway, and that it does exit when there is no unfinished work remaining.

The work class is copy-\/constructible so that it may be used as a data member in a handler class. It is not assignable.
